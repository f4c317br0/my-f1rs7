# Рекурсивные алгоритмы на Python

Этот проект содержит набор рекурсивных функций, решающих различные задачи: от математических вычислений и работы со строками до генерации фракталов и решения логических головоломок.

## 📁 Структура проекта

### 1. Математические задачи

#### `power.py` — Проверка степени числа
Рекурсивная функция `power(number, base)`, проверяющая, является ли число `number` степенью числа `base`.
- **Логика**: Делит число на основание, пока оно не станет равным 1 (успех) или не перестанет делиться без остатка (неудача).
- **Пример**: `power(81, 3)` → `True` ($3^4 = 81$).

#### `convert.py` — Перевод систем счисления
Рекурсивная функция `convert(number, base)`, переводящая число из десятичной системы в любую другую (кроме унарной).
- **Возвращает**: Список "цифр" результата от старшего разряда к младшему.
- **Пример**: `convert(300, 16)` → `[1, 2, 12]` ($1 \cdot 16^2 + 2 \cdot 16^1 + 12 \cdot 16^0 = 300$).

#### `accumulate_sum.py` — Накопительная сумма
Рекурсивная функция `accumulate_sum(lst)`, заменяющая каждый элемент списка суммой всех предыдущих элементов включая текущий.
- **Пример**: `[1, 2, 3]` → `[1, 3, 6]`.

#### `paths.py` — Пути в Королевстве чисел
Рекурсивная функция `paths(a, b)` с мемоизацией, подсчитывающая количество путей от числа `a` к числу `b`.
- **Правила перехода**: $+1$, $+3$ или $\times 2$.
- **Пример**: `paths(1, 5)` → `7`.

---

### 2. Работа со строками

#### `is_palindrome.py` — Проверка палиндрома
Рекурсивная функция `is_palindrome(text)`, проверяющая строку на палиндром.
- **Особенности**: Игнорирует регистр и все символы, не являющиеся буквами.
- **Пример**: `'А роза упала на лапу Азора.'` → `True`.

---

### 3. Графика и фракталы (Pillow)
*Для работы этих файлов требуется библиотека `pillow` (`pip install pillow`).*

#### `bubbles.py` — Фрактал из пузырьков
Рекурсивная функция `bubbles(draw, side_length, radius, color, iterations, x, y)`.
- **Алгоритм**: Рисует круг, затем по вершинам ромба (верх, низ, лево, право) рисует 4 круга меньшего радиуса. Процесс повторяется рекурсивно.
- **Параметры**: Уменьшение стороны ромба в 3 раза, радиуса в 2 раза на каждом шаге.

#### `corners.py` — Вложенные квадраты
Рекурсивная функция `corners(draw, size, color, iterations, x, y)`.
- **Алгоритм**: Делит квадрат на 4 части. В верхнем левом и нижнем правом рисует внутренние квадраты с отступом 10px. Рекурсия применяется к этим двум новым областям.

---

### 4. Логические задачи

#### `sudoku.py` — Заполнение квадрата (Упрощенный Судоку)
Рекурсивная функция `sudoku()`, работающая с глобальной переменной `board`.
- **Задача**: Заполнить пустые клетки (`''`) первыми $N$ буквами алфавита так, чтобы в каждой строке и столбце буквы не повторялись.
- **Метод**: Backtracking (поиск с возвратом).

## 🚀 Как запустить

1. Убедитесь, что установлен Python 3.
2. Для графических задач установите библиотеку Pillow:
   ```bash
   pip install pillow
   ```
3. Запустите любой файл как скрипт (если в нем есть пример вызова) или импортируйте функции в свой код:
   ```python
   from power import power
   print(power(81, 3))
   ```
